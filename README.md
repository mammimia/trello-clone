# Trello Clone App

This is a Trello Clone App project. It was developed for learning purposes.

## Tech Stack

- Next.js 14
- Server Actions
- Prisma
- Tailwind
- Stripe
- MySQL

## Installation

To install this project follow the steps after cloning.

```bash
  npm install
```

Then to start app

```bash
  npm run dev
```
